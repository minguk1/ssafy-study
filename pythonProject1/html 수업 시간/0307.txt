< > : 태그

html : 문서의 최상위root 요소

head : 문서 메타데이터 요소
    문서 제목, 인코딩, 스타일, 외부 파일 로딩 등
    일반적으로 브라우저에 나타나지 않는 내용

body : 문서 본문 요소
    실제 화면 구성과 관련된 내용

<title> : 브라우저 상단 타이틀
<link> : 외부 리소스 연결 요소(CSS 파일 등)
<style> : CSS 직접 작성

여는 태그와 닫는 태그 반드시 구분
이름 같아야한다.
모든 내용은 태그로 감싸져 있어야 한다.

ex)    <h1>contents</h1>

inline 스타일은 사용 자제

span <-> div

lorem 쓸데 없는 텍스트들을 만들어준다.

css 적용 우선순위 알고 있어야 한다.
important 남용 금지
